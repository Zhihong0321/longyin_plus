## 本次更新
- 统一当前受支持的安装与启动流程，主入口固定为 `龙胤立志传 Pro Max` Electron 启动器
- 从活动载荷中移除旧版 `mod-prototype`、控制脚本、探针与追踪插件，只保留当前受支持插件并把历史内容归档
- Electron 设置页新增并直连对话快进、月度对话倍率、心悟、突破成功额外天赋等选项
- 启动器保存设置时改为直接写入主配置，并清理旧版遗留配置与插件文件
- 修复活动 `LongYinStaminaLock` 插件中的 `P` 对话快进，使其重新按 0.5 秒节奏脉冲推进并在遇到选项时自动停下
- 优化启动日志与打包配置，固定 `LongYinProMax` 可执行名，便于 OTA 包和本地启动一致

## 使用说明
- 旧版 `0.1.1` 启动器检查更新后，应可发现 `0.1.2`
- 更新后请继续通过 `龙胤立志传 Pro Max` 启动器安装、配置并启动游戏
- 本次发布会重新上传 OTA ZIP 与 `update-manifest.json`，更新日志直接读取 GitHub Release body
